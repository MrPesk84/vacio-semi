# Adrenalina — Web de deportes de riesgo (plantilla)

Pequeña página estática con HTML/CSS/JS. Archivos:

- `index.html` — página principal
- `estilos.css` — estilos modernos y sencillos
- `move.js` — interacciones (menú, smooth scroll, reveal, tilt, modal)

Cómo usar

1. Abrir `index.html` en tu navegador (doble clic o desde PowerShell):

```powershell
# desde la carpeta del proyecto
ii .\index.html
```

2. O abrir con tu editor/servidor de desarrollo favorito.

Notas y mejoras posibles

- Añadir optimización de imágenes y un sistema de assets local.
- Implementar fetch a un backend o integración con formularios reales.
- Más accesibilidad (focus management en modal, etc.).

Hecho con cariño para una UI limpia y responsiva.
